# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for the Luqma project, following the [MADR 4.0.0](https://adr.github.io/madr/) format.

## ADR List

- [ADR-0001](0001-use-markdown-architectural-decision-records.md) - Use Markdown Architectural Decision Records
- [ADR-0002](0002-use-monorepo-structure.md) - Use Monorepo Structure

## Creating a New ADR

1. Copy [`adr-template.md`](adr-template.md)
2. Name it `NNNN-title-with-dashes.md` (next sequential number)
3. Fill in all required sections with at least 2-3 considered options
4. Add the new ADR to the list above

Alternatively, ask Cursor AI: "Create an ADR for..." and it will handle everything automatically.
